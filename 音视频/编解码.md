1. **编码**
    - 编码的目的：为了压缩，各种视频编码方式，都是为了让视频体积变得更小，有利于存储和传输。编码的核心思想就是去除冗余信息。
2. **IPB帧：**
    - I帧：帧内编码帧，关键帧，可单独解码出一个完整图像
    - P帧：向前预测编码帧，记录了本帧跟之前的一个关键帧的差别
    - B帧：双向预测编码帧，记录了本帧与前后帧的差别
3. **H.264码流分析**
    - VCL（Video Coding Layer 视频编码层）
    - NAL（NetWork Abstraction Layer 网络提取层）
4. - H.264  --> AVC
   - H.265  --> HEVC
